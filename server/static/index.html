WebLocate API

WebLocate is an API for searching domain and IP information from over the web. This API will find IP details, location details and business details of the responsible hosting company.

API End Points

  • Domain Search: search/domain/:domain
  • Email Domain Search: search/domain/:email
  • IP Search Example: search/ip/:ip
  • Counter Read Example: counter/read
  • Counter Update Example: counter/update

API Examples

  • Domain Search Example: /search/domain/www.google.com
  • Email Domain Search Example: /search/domain/someone@outlook.com
  • IP Search Example: /search/ip/8.8.8.8
  • Counter Display Example: /counter/read
  • Counter Update Example: /counter/update
